**VR Lab Class Models README**

In this folder you will find three packages containing 3D models which you can use for the development during the VR Lab class.\
**Important:** The models are intended for use within the Lab Class only. Any use outside of this is prohibited.

You can find the respective models for the different themes under the following paths:

- **Fantasy**: Assets/VR Lab Class Models/LowPolyDungeons/Prefabs
- **Office**: Assets/VR lab Class Models/PolygonOffice/Prefabs
- **Sci-Fi**: Assets/VR Lab Class Models/PolygonSciFiSpace/Prefabs
